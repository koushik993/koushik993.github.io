Jack-the smart stat



This is a prototype-app for a new thermostat UI that does not rely on numbers.
Features:
Instead of temperature setpoints, the system relies on user inputting their current comfort level ( feeling hot, warm, slightly warm, cold, slightly cool etc.) which is a more intuitive way of expressing how comfortable one feels.
Created a personable avatar system "Jack", as a living character, a home butler, who is learning your comfort.
UI design is more visual with an avatar display and image buttons.
Usage of visual avatar animations (throwing wood into the fireplace, and extinguishing the fire) and audio (bell and gong) for confirmation of user input.
Created an option to create weekly and temporary schedules.
A simple color-coded calendar view for setting schedules.
A tutorial explaining the buttons and concept of this new thermostat design.
